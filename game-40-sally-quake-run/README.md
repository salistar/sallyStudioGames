# 40 Quake Run

This is the 40 Quake Run mobile application, part of the Sally Games suite.


## 🎯 Version Implementation Roadmap
- **v1.0.0 (MVP Release):** Core gameplay loop, basic Sally Theme, and local storage.
- **v1.1.0 (Monetization):** AdMob Integration (Banner, Interstitials, Rewarded Video).
- **v1.2.0 (Localization):** Arabic, Darija, French, and English support.
- **v2.0.0 (Enhanced UI):** Animations, dynamic backgrounds, and sound design.
- **v2.5.0 (Social):** Leaderboards and achievements.
- **v3.0.0 (Cross-Platform):** iOS release and Web support.
